About Myself

I have created this website so that any visitor can learn more about myself and the kind of services that I can provide. I am a full stack engineer that is passionate about creating applications that have the best user experience possible.

I started my journey into software development because of a realization: that I was using computers and the web on a daily basis, but I had no real knowledge of how either worked under the hood. My curiosity eventually led me to realize that I really enjoy learning about and implementing software.

Since then I've devoted my full time to developing software and my skills with it. When I design applications, I want them to look striking, be intuitive to use, and have all the functionality that a user would expect. The projects I've showcased on this website encapsulate these principles.

My interests outside of software include board and video gaming, fish keeping, and spoiling my cat Bingsu!
